# Corporate Job Simulations Portfolio 💼📈

Welcome to my portfolio! This repository showcases a collection of hands-on data analytics and risk management simulations completed with global industry leaders. These projects allowed me to apply technical logic to solve real-world corporate problems.

* **Objective:** Acted as a Data Analyst to assist executive leadership (CEO & CMO) with a global market expansion strategy.
* **Core Tasks:** Cleaned raw online retail transaction data, resolved inconsistent date formats using logic, and filtered out anomalies. Developed executive-level visuals (Time-Series, Top 10 breakdowns, and global Map Charts).
* **Folder Link:** [View TATA Project](./01-TATA-Data-Visualisation/)
